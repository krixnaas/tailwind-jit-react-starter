# Tailwind JIT Starter React

## Setup

1. Clone repo

```bash
git clone https://github.com/krixnaas/tailwind-jit-react-starter.git
```

2. Install Packages

```bash
npm i
```

3. Watch Css

```bash
npm run watch:css
```

4. Start Server

```bash
npm start
```
